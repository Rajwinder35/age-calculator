- Track performance objectively
- Identify patterns in wins/losses
- Improve decision-making
- Build discipline and accountability

## Quick Reference Checklist

### Before Every Trade:
- [ ] Signal confirmed on chart
- [ ] Volume spike present
- [ ] Active trading session
- [ ] Trend aligned
- [ ] Position size calculated (2% risk)
- [ ] Stop loss determined
- [ ] Take profit calculated (2.5:1)
- [ ] Daily loss limit not reached
- [ ] Emotional state is calm
- [ ] No major news in next 30 minutes

### After Every Trade:
- [ ] Record entry details
- [ ] Take screenshot
- [ ] Note emotional state
- [ ] Record exit details
- [ ] Analyze what happened
- [ ] Update statistics
- [ ] Learn from the trade

### End of Day:
- [ ] Calculate daily P&L
- [ ] Review all trades
- [ ] Update journal
- [ ] Identify patterns
- [ ] Plan for tomorrow

### End of Week:
- [ ] Calculate weekly stats
- [ ] Review performance
- [ ] Identify strengths/weaknesses
- [ ] Set goals for next week
- [ ] Adjust strategy if needed
